Key features

2 Modes: Active and Passive

You can actively analyze every single web request your browser makes and individually allow or deny requests. What you allow or deny can also be passively enforced during your entire browsing experience.

Custom Rule sets

You can create rules using regular expressions that tells MIMTool to allow or deny URLs. These rules are always enforced, in both active and passive modes.

Tracked History

See a history of all requests you have allowed or denied. Reorder these to change which rules are hit first. Remove rules you do not want anymore.

Preview window

See the contents of a response to a request before the browser can render it! See the raw text of the HTML/CSS/JS, preview the PNG/JPEG/GIF, and see a hexdump of unknown binary file before they can cause any harm.
